# Awesome-Chocolates-Sales-Power BI-Dashboard



This project is a Power BI Sales Performance Dashboard created for Awesome Chocolates, showcasing key metrics and trends for data-driven decision-making.

![sale analysis](https://github.com/user-attachments/assets/7869f128-e786-4616-a58b-b91f5320f764)



---

##  Key Features

- **Total Sales:** $34M  
- **Total Profit:** $20M  
- **Total Costs:** $13M  
- **Boxes Sold:** 2M  
- **Shipments:** 6K  
- **Profit Percentage:** 60.3%

---

##  Visualizations Included

- KPI Tiles with Month-over-Month (MoM) comparisons  
- Line Chart for monthly sales trend  
- Shipment Analysis using histogram and KPI gauge  
- Sales by Salesperson with Profit and LBS%  
- Profit % Indicator Dial

---

##  Tools Used

- Power BI
- DAX for Calculated Measures
- Data Modeling using Sales, Costs, Products, and Shipment Tables

---

##  Insights

- Identifies top-performing salespersons
- Highlights shipment vs. sales correlation
- Tracks monthly trends to monitor business growth
